# React_Redux_Webpack_Babel_Chunk_JS_CSS_Start
使用React、Redux、Webpack、Babel（ES6）（脚手架），Webpack按development和production环境配置好了有关热更新、postcss、hash命名（缓存方案）、UglifyJs等config。
按需加载JS、按需加载CSS

### INSTALL:
```npm install```
### DEVELOPMENT:
```npm start```

``` open http://localhost:8080/ ```

### PRODUCTION:
```npm run build```
